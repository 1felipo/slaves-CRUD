# Slaves CRUD
A CRUD where you will be able to manage your slaves jijiji!
## Available scripts
Install the project modules: in the project directory:

    npm install
run the project in development mode:

    npm run dev
build the project in "./dist":

    npm run build
remove the "./dist" folder:

    npm run clean
